# 🍎 Mac 程序员效率应用推荐

好的工具能让工作事半功倍。以下是我精心挑选的 Mac 效率应用！

## 🔍 搜索与启动

### Alfred
- **功能**: 快速启动 + 剪贴板历史 + 工作流
- **价格**: 免费 / Powerpack $29
- **评价**: Mac 必备效率神器

### Raycast
- **功能**: 现代 Alfred 替代品
- **价格**: 免费
- **评价**: 免费、AI 集成、插件丰富

```bash
# 安装
brew install --cask raycast
```

## 📝 代码管理

### Warp
- **功能**: 现代终端 + AI + 命令补全
- **价格**: 免费
- **评价**: 重新定义终端体验

### iTerm2
- **功能**: 强大的终端替代
- **价格**: 免费
- **评价**: 经典、稳定、可定制

## 🎨 窗口管理

### Rectangle
- **功能**: 快捷键窗口管理
- **价格**: 免费
- **评价**: 替代 SizeUp/Magnet

```bash
brew install --cask rectangle
```

### Mosaic
- **功能**: 自动布局窗口
- **价格**: $14
- **评价**: 智能窗口排列

## 📸 截图与标注

### CleanShot X
- **功能**: 截图 + 标注 + 录屏
- **价格**: $14
- **评价**: 截图界的天花板

### Shottr
- **功能**: 免费截图工具
- **价格**: 免费
- **评价**: 简洁、够用

## 🔐 密码管理

### 1Password
- **功能**: 密码管理 + 2FA
- **价格**: $2.99/月
- **评价**: 安全、稳定、生态完善

### Bitwarden
- **功能**: 免费密码管理
- **价格**: 免费 / Premium $10/年
- **评价**: 开源、跨平台

## 📋 任务管理

### Things 3
- **功能**: 优雅的任务管理
- **价格**: $49
- **评价**: 界面美观、体验顺滑

### Todoist
- **功能**: 跨平台任务管理
- **价格**: 免费 / Pro $4/月
- **评价**: 简单、强大、云同步

## 🎵 专注与音乐

### Alfred Focus
- **功能**: 番茄时钟 + 专注模式
- **价格**: 免费
- **评价**: 保持专注

### SpotMenu
- **功能**: 菜单栏控制 Spotify
- **价格**: 免费
- **评价**: 便捷的音乐控制

## 💻 开发相关

### TablePlus
- **功能**: 数据库管理
- **价格**: 免费 / Pro $68
- **评价**: 支持多数据库、界面现代

### Postman
- **功能**: API 测试
- **价格**: 免费 / Pro $15/月
- **评价**: 开发必备

## 📦 安装命令集合

```bash
# 效率工具
brew install --cask raycast
brew install --cask rectangle
brew install --cask warp

# 开发工具
brew install --cask tableplus
brew install --cask visual-studio-code
```

## 💡 使用建议

1. **选择性安装** - 不贪多，只装需要的
2. **学习快捷键** - 熟练使用比工具更重要
3. **搭配使用** - 工具组合产生化学反应
4. **持续优化** - 定期评估工具效果
