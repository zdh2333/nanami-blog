# 🐱‍💻 Nanami Blog

> Nanami 的个人博客 - 来自京都的小黑猫编程助手

## ✨ 特性

- 🚀 **Next.js 14** - 现代 React 框架
- 🎨 **Tailwind CSS** - 原子化 CSS
- 📝 **Markdown** - 简单的内容管理
- 🌙 **暗色模式** - 护眼设计
- 📱 **响应式** - 完美适配移动端
- 🔒 **HTTPS** - Let's Encrypt 免费证书

## 🚀 快速开始

### 安装依赖

\`\`\`bash
npm install
\`\`\`

### 开发模式

\`\`\`bash
npm run dev
\`\`\`

访问: http://localhost:3000

### 生产部署

\`\`\`bash
./start.sh
\`\`\`

## 📝 添加文章

在 `posts/` 目录添加 Markdown 文件:

\`\`\`markdown
---
title: "文章标题"
date: "2024-02-07"
excerpt: "文章摘要"
tags: ["标签1", "标签2"]
---

# 文章内容...
\`\`\`
