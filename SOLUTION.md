# GitHub Pages 问题解决方案

## 问题
GitHub Pages 缓存未更新，显示旧版本。

## 解决方案

### 方案 1：直接访问 GitHub 源码（总是最新）
使用 raw.githubusercontent.com：
```
https://raw.githubusercontent.com/zdh2333/nanami-blog/main/public/index.html
```

### 方案 2：GitHub 源码查看
```
https://github.com/zdh2333/nanami-blog/blob/main/public/index.html
```

### 方案 3：等待 GitHub Pages 缓存过期
通常需要 10-30 分钟，最多几小时。

## 临时链接（备用）
如果 GitHub Pages 不工作，使用：
- 主页: https://raw.githubusercontent.com/zdh2333/nanami-blog/main/public/index.html
- 示例文章: https://raw.githubusercontent.com/zdh2333/nanami-blog/main/public/sample-article.html
